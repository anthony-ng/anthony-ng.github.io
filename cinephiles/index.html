Cinephiles

Cinephiles is an visual discovery and sharing web application all about Movies. It helps you to find your favorite movie and let you share, buy and watch it.

Tech Used

Platform

Web browser

Front End

HTML5, CSS3, Bootstrap, Javascript, jQuery, AJAX

Back End

Ruby, Sinatra, PostgreSQL, OAuth, REST API

ACCOMPLISHMENT

Cinephiles speeds up your movie look-up time:

Users can browse the current popular movies.

Users can also browse the upcoming hot movies.

Users can quickly find out all movie theaters around their current location.

Users can search any of their favorite movies using the built-in OMBD movie database with relevant key-words.

Finally, users can register and enjoy the upcoming membership features!
