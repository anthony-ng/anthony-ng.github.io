Connect Four

Connect Four is a two-player connection game in which the players first choose a color and then take turns dropping colored discs from the top into a seven-column, six-row vertically suspended grid. The objective of the game is to connect four of one's own discs of the same color next to each other vertically, horizontally, or diagonally before your opponent.

Tech Used

Platform

Web browser

Front End

HTML5, CSS3, Bootstrap, Javascript, jQuery

Back End

Firebase.js

ACCOMPLISHMENT

We built a fully functional, JavaScript-heavy web application from scratch using agile development techniques:

Players can see a styled board.

Players can place a piece into a column.

Players are notified when they win the game

Players can play against each other in the same browser.

And, players can play against another user on a different computer, powered by Firebase.js
